# 规则集模板

> 复制此文件创建新规则集，填写 6 个章节后重命名保存至 `references/rulesets/`。
> 然后在框架初始化时选择该规则集，或通过 `加载规则集 <名称>` 手动加载。

---

## 1. 元信息

```yaml
名称: （你的规则集名称）
适用流派: （都市/修仙/赛博/西幻/历史/悬疑等）
约束强度: （宽松/中等/严格）
简介: （一句话说明本规则集的审查重点）
```

## 2. 能力因果法则

能力使用必须遵循的逻辑链步骤。填写本流派的典型能力使用路径：

```
例·赛博: 侦测 → 寻找漏洞 → 算力推演 → 夺取权限
例·修仙: 运气 → 掐诀 → 法力流转 → 法术释放
例·都市: 观察 → 推理 → 行动
本流派: （在此填写）
```

**禁止行为**：结果与能力设定不符的跳步。

## 3. 代价显性法则

不同等级能力对应的典型代价。至少填写小/中/大三级。

| 能力级别 | 典型代价 | 必须描写的生理/环境反馈 |
|:--------|:---------|:----------------------|
| 小 | （例：精神力轻度消耗 / 轻微体力流失） | （例：额角微汗 / 呼吸加快） |
| 中 | （例：灵力池消耗过半 / 连续操作超 4 小时） | （例：手抖 / 视线模糊 / 面色发白） |
| 大 | （例：境界跌落 / 法器损毁 / 烧毁硬件） | （例：内伤呕血 / 核心组件冒烟 / 毁灭性损耗） |

## 4. 空间连续法则

本流派典型移动方式及合理耗时参考。

| 方式 | 耗时基准 | 本流派替代方案 |
|:----|:---------|:--------------|
| 短距离（同城） | 步行 10-30 分钟 / 车 10-20 分钟 | （例：御剑飞行 5 分钟） |
| 中距离（跨境） | 2-8 小时 | （例：传送阵 10 秒 + 冷却 2 小时） |
| 长距离（跨大陆） | 1-7 天 | （例：灵舟航行 3-10 天） |

## 5. 因果不可逆法则

定义本流派中哪些事件被视为不可逆。

- 死亡：□ 绝对不可逆 □ 可复活但有严格代价（代价：\_\_\_\_\_\_）
- 力量损毁：□ 绝对不可逆 □ 可修复但有严格条件（条件：\_\_\_\_\_\_）
- 身份暴露：□ 一次即永久 □ 可挽回（方式：\_\_\_\_\_\_）
- 关系破裂：□ 不可逆 □ 可修复（方式：\_\_\_\_\_\_）

## 6. 资源约束追踪

本流派的资源池字段清单及实时工况字段。这些字段会出现在 YAML 链条的第 3 区块。

**资源池字段**：
```yaml
例·赛博: VRAM 总池 / 当前占用 / 核心温度
例·修仙: 灵力池上限 / 当前灵力 / 境界突破状态
你定义:
```

**实时工况字段**：
```yaml
例·赛博: 核心温度阵列平均 / 最高单卡 / 冷却液位
例·修仙: 经脉通畅度 / 灵气浓度 / 丹田压力
你定义:
```
