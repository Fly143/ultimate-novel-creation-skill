# 规则集：通用默认

> 维度齐全，约束宽松。适合多数写作场景的通用规则集。
> 不绑定任何流派特定的名词或设定。

---

## 1. 元信息

```yaml
名称: 通用默认
适用流派: 通用（不限定）
约束强度: 宽松
简介: 覆盖五维审查的基础规则集。所有维度都设定了底线约束，但阈值宽松，不会对多数写作风格产生限制。
```

## 2. 能力因果法则

能力使用必须遵循的基本逻辑链：

```
动机 → 判断 → 行动 → 结果
```

**禁止行为**：
- 主角在不具备对应能力的情况下直接解决问题（空降解围）
- 之前从未展示过的能力在毫无铺垫的情况下出现
- 结果与行动之间缺少合理的因果桥接

## 3. 代价显性法则

| 能力级别 | 典型代价 | 必须描写的生理/环境反馈 |
|:--------|:---------|:----------------------|
| 小 | 时间消耗（数分钟到数小时）/ 少量体力或精神消耗 | 呼吸变化 / 轻微的体力信号 |
| 中 | 显著的能量/资源消耗 / 连续操作超过临界时长 | 疲劳表现（出汗、手抖、面色变化） |
| 大 | 关键资源损毁 / 不可逆损耗 / 重大代价支付 | 毁灭性的物理或精神反馈 |

## 4. 空间连续法则

| 距离 | 耗时基准 |
|:----|:---------|
| 短距离（同城/同区） | 步行 10-30 分钟 / 交通工具 10-20 分钟 |
| 中距离（跨城） | 1-6 小时 |
| 长距离（跨国/跨大陆） | 1-14 天 |
| 远距离（跨世界/跨位面） | 必须有明确的通行方式和时间描述 |

## 5. 因果不可逆法则

- **死亡**：主配角死亡绝对不可逆。如有复活设定，必须世界规则中明确允许且主角/配角付出了对等代价。
- **关键损毁**：关键物品/能力的损毁不可自动恢复。恢复需要明确描写修复过程（时间、工具、方法）。
- **身份暴露**：暗面身份暴露后，剧情必须在合理的时间窗口内展现后果。
- **关系变化**：重大关系破裂后，除非双方都付出努力，否则不可自动修复。

## 6. 资源约束追踪

**资源池字段**：
```yaml
- 金钱/财富储备
- 关键物品/装备状态
- 人际关系/声望（如有）
```

**实时工况字段**：
```yaml
- 当前疲劳度 / 连续清醒时长
- 主要资源（钱/装备/人脉）的最低保有量
```
