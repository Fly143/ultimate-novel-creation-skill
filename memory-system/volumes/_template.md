# 卷压缩 — 模板

> **每完成 5 个阶段（50 章），将 5 个 phase 压缩为本文件。  
> 文件命名：`volume_{序号}_ch{起始}-{结束}.md`，如 `volume_1_ch1-50.md`  
> 这是全书最高层级的记忆缓存——一个 volume 就是半本书的「一句话历史」。**
