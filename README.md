# 全能小说作家

[![Version](https://img.shields.io/badge/version-5.0.0-blue)](https://github.com/Fly143/ultimate-novel-creation-skill)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

面向 AI Agent 的全流程长篇小说创作技能。涵盖书名生成、世界观构建、人设设计、情节大纲、章节写作、爽点设计、节奏诊断，配备自然语言记忆系统确保长篇连贯性，支持 AI 全自动和分步引导双模式。

## 核心能力

- **22 个功能模块**：从项目初始化到导出发布全覆盖
- **四层自然语言记忆系统**：故事圣经 / 人物弧线 / 剧情时间线 / 伏笔清单，不依赖外部工具
- **千章级自适应上下文**：章节类型判定（高潮/伏笔回收/过渡/标准），自动调配上下文加载深度
- **远古伏笔语境回读**：第 500 章回收第 1 章伏笔时，自动回读原始章节摘要保证语境完整
- **Bible 文档增长控制**：超 30KB 自动压缩远古内容，细节下沉到卷压缩，按需回读
- **双轨书名策略**：网文风（起点/番茄）+ 出版风（实体/IP），按目标平台切换
- **40+ 写作参考**：对话技巧、打斗场景、情感描写、爽点设计、节奏诊断、毒舌/神回复风格等
- **16 个实用模板**：人设卡、人物关系图谱、情节大纲、章节规划、百万字规划、节奏诊断报告等

## 项目结构

```
├── SKILL.md              # 技能入口，路径约定
├── modules/              # 28 个功能模块（执行协议 → 启动引导 → 22 功能 → 场景映射）
├── references/           # 40+ 写作技法参考
├── templates/            # 16 个输出模板
└── memory-system/        # 记忆系统模板（skill 内只读，项目初始化时复制）
    ├── bible/            # 故事圣经 / 人物弧线 / 剧情时间线 / 伏笔清单
    ├── summaries/        # 单章摘要模板
    ├── phases/           # 阶段压缩模板（每 10 章）
    └── volumes/          # 卷压缩模板（每 50 章）
```

## 快速开始

将本仓库作为 AI Agent 的技能加载。Agent 加载后会自动读取 `SKILL.md` 及相关模块。

用户输入「我想写小说」即可触发完整创作流程。
