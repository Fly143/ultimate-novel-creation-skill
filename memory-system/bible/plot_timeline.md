## 时间线一致性检查

> 每次新章完成后检查以下问题，**避免时间线冲突**：

- [ ] 本章的时间点（季节/日期/时辰）与前章衔接合理？
- [ ] 人物没有同时出现在两个地方？
- [ ] 已死/已退场的角色没有被错误地写回来？
- [ ] 重大事件之间的时间跨度合理？
- [ ] 角色的年龄/成长速度一致？
