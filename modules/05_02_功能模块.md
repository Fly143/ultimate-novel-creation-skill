### 2. 世界观构建 

**功能**：构建世界框架、力量体系、规则设定，同步到记忆系统
**参考资源**：
**模板**：`templates/世界观设定文档.md`
**记忆系统**：完成后自动同步到 `[书名]/记忆系统/bible/story_bible.md`（世界观简版 + 阶段划分）

**执行流程（强制按顺序执行）**：
```
=== 世界观构建阶段 ===
1. 确定世界类型（古代/现代/未来/架空）
2. 设计核心规则体系（力量体系、社会规则、特殊设定）
3. 构建势力分布和地理环境
4. 保存世界观设定：[书名]/设定/[书名]_世界观设定.md

=== 记忆系统同步（如已初始化） ===
5. 【检查】确认 [书名]/记忆系统/bible/story_bible.md 是否存在
6. 【同步】将世界观要点写入 story_bible.md：
   - 更新「世界观简版」段落
   - 更新「阶段划分」中的开篇阶段描述
   - 如有力量体系等级，记录在故事圣经中
```

**输入信息**：
- 世界类型（古代/现代/未来/架空）
- 核心规则设定
- 关键势力分布

**输出**：
- 世界观设定文档：`[书名]/设定/[书名]_世界观设定.md`
