### 10. 打斗场景创作

**功能**：设计打斗结构、详细描写招式、心理博弈、金手指运用
**参考资源**：`references/打斗场景创作指南.md`
**模板**：无专用模板

**记忆系统**：打斗场景涉及实力变化时，同步到 `[书名]/记忆系统/bible/character_arcs.md`；关键战斗事件同步到 `[书名]/记忆系统/bible/plot_timeline.md`

---

**执行流程**：

```
=== 打斗场景创作阶段 ===
1. 确认打斗类型（碾压型/苦战型/越级挑战型/势均力敌型）
2. 设计双方实力对比
3. 读取 `references/打斗场景创作指南.md`
4. 确定金手指参与方式
5. 按结构创作：试探 → 交锋 → 僵持 → 突破 → 决胜
6. 加入爽点元素（如需要）
```

**输入信息**：
- 打斗类型（碾压型/苦战型/越级挑战型/势均力敌型）
- 双方实力对比
- 金手指参与方式
- 打斗目的和结果

**输出要点**：
- 标准打斗结构：试探 → 交锋 → 僵持 → 突破 → 决胜
- 爽点打斗结构：反派嚣张 → 主角冷静 → 激烈交锋 → 金手指运用 → 反派被碾压 → 众人震惊

**输出**：
- 打斗场景内容（融入章节）
