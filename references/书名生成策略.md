# 书名生成策略

## 书名是一句广告，不是一个总结

好书名让目标读者在 0.5 秒内决定「点进去看看」。坏书名要么什么都没说，要么什么都说了但像关键词堆砌。

书名的敌人不是「太长」或「太短」，是**没钩子**。

---

## 双轨生成：先判平台，再定策略

### 🅰️ 网文风（起点/番茄/七猫）

读者靠搜索标签和推荐流找书。书名必须有**信息量 + 钩子**。

**生成方法（按优先级）**：

1. **提取三要素**：题材 + 初始状态 + 金手指/能力
   → 《修仙：开局杂灵根，我吞了诸天》

2. **制造反差**：两个互相矛盾的词放在一起
   → 《废柴？你管这叫废物？》

3. **承诺终点**：直接告诉读者主角会成为什么
   → 《从杂役到仙帝》

4. **展示规则**：世界观里最特别的那条规则
   → 《全球御兽：只有我知道进化路线》

**AI 容易犯的错**：
- 堆标签：「都市神豪系统无敌逆袭」——每个词都是标签，合起来什么都没说
- 太老实：《逆命仙途》——能猜出是修仙，但和一千本修仙书放在一起毫无区别
- 套公式过头：《重生之XXX》《我只想XXX》——太像模板，读者直接划过

**自检**：如果删掉书名里的冒号和「之」，它还成立吗？不成立就重想。

### 🅱️ 出版风（实体书/IP向）

读者在书店或推荐列表里浏览。书名需要**留白 + 想象空间**。

**生成方法**：

1. **以核心矛盾命名**：把故事里最根本的冲突压缩成短语
   → 《凡人修仙传》——「凡人」+「修仙」= 天生力量悬殊

2. **以核心概念命名**：一个词承载整个世界观
   → 《诡秘之主》——不说「克苏鲁」「序列」「非凡者」，三个字就够了

3. **以人物命名**：但人物名本身要有信息量
   → 《甄嬛传》——「嬛」字暗示女性+宫廷，读音婉转

4. **以气质命名**：短词，不解释，靠气质吸引
   → 《活着》《间客》——不说什么类型，但读者感觉得到重量

**AI 容易犯的错**：
- 太「文艺」但空洞：《时光之尘》《星辰彼岸》——漂亮但没有故事
- 仿经典但东施：《诡秘至尊》《三体纪元》——搭便车会被一眼看穿

---

## 生成流程（不是选模板，是迭代）

1. **提取核心钩子**：这个故事最特别的一个设定、一个矛盾或一个人物特质是什么？用一句话说出来。
2. **生成 10 个草稿**：不要在第一轮就筛选。先量产。两套风格各出 5 个。
3. **筛掉 5 个**：删掉任何和同类书撞名的、任何需要解释才能懂的、任何读三遍还记不住的。
4. **剩下 5 个中挑 3 个给用户**：解释每个的定位（网文风还是出版风，为什么有效），让用户选。

## 终极测试

把候选书名念给一个没看过小说简介的人听。问 ta：你觉得这是什么类型的故事？

如果 ta 猜对了——及格。如果 ta 追问「讲什么的」——及格以上。如果 ta 说「给我看看」——赢了。
