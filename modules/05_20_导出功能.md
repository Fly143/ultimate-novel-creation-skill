# 功能模块 20：导出功能

**功能**：将小说内容导出为不同格式，适配各平台需求。

---

## 支持的导出格式

| 格式 | 用途 | 文件类型 |
|------|------|----------|
| 网文平台 | 起点、番茄、七猫等 | .txt |
| 电子书 | Kindle、阅读APP | .epub |
| Word文档 | 编辑、投稿 | .docx |
| Markdown | 笔记、备份 | .md |

---

## 执行流程

### 1. 网文平台导出

```
输入：导出网文格式，第1-10章
输出：
- 合并为单个txt文件
- 文件名：[书名]_第1-10章_网文版.txt
- 格式：纯文本，章节间空行分隔
```

### 2. 电子书导出

```
输入：导出epub格式
输出：
- 生成.epub文件
- 包含目录导航
- 文件名：[书名].epub
```

### 3. Word文档导出

```
输入：导出Word格式，投稿用
输出：
- 生成.docx文件
- 标准投稿格式（宋体小四，1.5倍行距）
- 文件名：[书名]_投稿版.docx
```

---

## 导出规范

- **章节范围**：可指定导出哪些章节
- **包含设定**：可选是否包含人设、大纲等设定文件
- **批量导出**：支持一次导出多个格式

---

## 快捷命令

- "导出网文" → 网文平台格式
- "导出epub" → 电子书格式
- "导出Word" → Word文档格式
- "导出全部" → 所有格式打包
