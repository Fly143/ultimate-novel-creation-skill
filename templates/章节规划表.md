# 章节规划表

| 章 | 标题 | 字数 | 类型 | 爽点 | 出场角色 | 场景 | 冲突目标 | 状态 |
|----|------|------|------|------|----------|------|----------|------|
| 1 | | | 日常/冲突/转折/高潮/过渡 | | | | | ⏳ |

> 类型判定：日常（铺垫）、冲突（矛盾建立）、转折（金手指/关键事件）、高潮（爽点释放）、过渡（承上启下）

> 出场角色：本章必须出现的角色（蓝图校验用）。场景：本章发生的地点。冲突目标：本章要推进的冲突 ID 或描述。
> **每章写作前自问**：
> - 读者此刻在等什么？（控制情绪缺口）
> - 本章必须兑现什么？暂不掀什么？
> - 章尾必须发生什么改变？（信息/关系/物理/实力至少一项）
> - 本章不要做什么？（硬禁止项）
