## 避免模板化词汇

### 常见模板词汇

**情绪类模板词**：
- 充满了、坚定、不甘、难以置信、震惊、愤怒、绝望、悲伤

**状态类模板词**：
- 冷冷地说、愤怒地说、嘲讽地说、轻蔑地说、温柔地说、平静地说、激动地说

**动作类模板词**：
- 紧紧握住、狠狠地、猛地、傻傻地、默默地

### 替代方案

**替代原则**：
1. 用具体动作代替抽象词汇
2. 用场景细节代替情绪描述
3. 用对话代替旁白
4. 用对比代替直接描述

**示例对比**：

**错误示例**：
```
林风眼中充满了愤怒，紧紧握住拳头，不甘地看着赵峰。
```

**正确示例**：
```
林风盯着赵峰，拳头慢慢收紧，指节发白。
他呼吸变得急促，但没有说话。
```

### 模板词替代库

#### 愤怒类

| 模板词 | 替代方案 |
|--------|----------|
| 眼中充满愤怒 | 瞳孔收缩、眼神锐利、盯着对方不动 |
| 愤怒地说 | 声音从牙缝挤出、每个字都像在咬 |
| 狠狠地握拳 | 指节发白、青筋暴起、指骨咯吱作响 |
| 愤怒地瞪着 | 眼睛充血、目光如刀、死死盯着 |

**实战示例**：
```
❌ 林风愤怒地看着赵峰，眼中充满了杀意。

✓ 林风盯着赵峰，瞳孔微微收缩。
他的手搭在剑柄上，没有说话。
空气中弥漫着一股寒意。
```

#### 悲伤类

| 模板词 | 替代方案 |
|--------|----------|
| 眼中充满悲伤 | 眼眶发红、眼角有水痕、眼神空洞 |
| 悲伤地说 | 声音发颤、说到一半停住、声音越来越轻 |
| 难过地低下头 | 肩膀下沉、整个人像泄了气、目光呆滞 |

**实战示例**：
```
❌ 苏晴悲伤地看着远方，眼泪从脸上滑落。

✓ 苏晴站在窗边，看着楼下的车流。
她的眼睛红了，眼角有一道水痕。
她没有擦，任由它滑落到下巴。
```

#### 紧张类

| 模板词 | 替代方案 |
|--------|----------|
| 充满了紧张 | 手心出汗、喉结滚动、不自觉地握紧 |
| 紧张地看着 | 眼神游移、不敢直视、频频看表 |
| 紧张地说 | 声音发紧、话说到一半卡住、语速变快 |

**实战示例**：
```
❌ 顾沉很紧张，不停地看时间。

✓ 顾沉站在门口，每隔几秒就看一次手机。
屏幕亮了，他立刻拿起，又放下。
如此反复十次。
```

## Show, Don't Tell原则

### 核心要求

- 用动作展示情绪，而不是直接告诉读者
- 用场景营造氛围，而不是解释说明
- 用细节暗示状态，而不是直接描述

### 五种实现方法

#### 方法1：情绪→动作转化

**错误示例**：
```
林风很愤怒。
```

**正确示例**：
```
林风一脚踹翻了面前的椅子。
椅子撞在墙上，发出巨响。
他喘着粗气，盯着赵峰。
```

#### 方法2：氛围→环境转化

**错误示例**：
```
这是一个悲伤的夜晚。
```

**正确示例**：
```
窗外下着雨，雨水拍打在窗户上。
房间里很安静，只有苏晴的抽泣声。
灯关着，黑暗中只有手机屏幕的微光。
```

#### 方法3：状态→细节转化

**错误示例**：
```
苏晴很紧张。
```

**正确示例**：
```
苏晴坐在沙发上，手指不停地搓着衣角。
她看了一眼钟，又看了一眼门。
来回看了三次。
```

#### 方法4：性格→行为转化

**错误示例**：
```
赵峰很傲慢。
```

**正确示例**：
```
赵峰靠在椅子上，一只脚搭在桌子上。
林风站在他对面，低着头，不敢看他。
赵峰嗤笑一声，拿起桌上的茶杯。
```

#### 方法5：关系→互动转化

**错误示例**：
```
他们之间的感情很深厚。
```

**正确示例**：
```
林风看了一眼苏晴，苏晴正好也看向他。
两人对视一秒，都笑了。
"走吧。"林风伸出手。
苏晴握住他的手，"嗯。"
```

## 口语化对话技巧

### 核心技巧

#### 技巧1：停顿

**作用**：模拟真实对话的节奏

**示例**：
```
❌ "我不会放过你的。"

✓ "我不会，放过你的。"
或：
"我不会放过你的..."
```

#### 技巧2：省略

**作用**：留白，增加真实感

**示例**：
```
❌ "我真的很喜欢你，你能不能和我在一起。"

✓ "我真的很喜欢你..."
"嗯。"
"你能不能..."
"什么？"
"没什么。"
```

#### 技巧3：断句

**作用**：调整节奏，突出重点

**示例**：
```
❌ "你为什么要这么做？"

✓ "你，为什么要这么做？"
或：
"你为什么要，这么做？"
```

#### 技巧4：语气词

**作用**：增加口语感

**示例**：
```
❌ "好的。"

✓ "嗯。"
"好。"
"行吧。"
"行啊。"
"呵。"
```

#### 技巧5：口头禅

**作用**：体现人物性格

**示例**：
```
霸道总裁：嗯，好，滚。
傲娇角色：谁，我才不是。
热血角色：我一定要！
腹黑角色：哦，是吗。
```

### 对话场景对比

#### 场景1：表白

**错误**：
```
"苏晴，我喜欢你，你能不能和我在一起？"
"好的。"
```

**正确**：
```
"苏晴..."
"嗯？"
"我..."
"什么？"
"算了。"
"你说啊。"
"我，喜欢你。"
苏晴愣住了。
"你..."
"嗯。"
"嗯什么？"
"我也喜欢你。"
```

#### 场景2：冲突

**错误**：
```
"你为什么这么做？"
"我只是想帮你。"
"你这是在害我。"
```

**正确**：
```
"你为什么要这么做？"
"我只是想帮你。"
"帮我？"
"嗯。"
"你这是在害我！"
"我..."
"够了。"
```

## 细节表现情绪

### 动作细节库

#### 愤怒的表现

```
林风盯着赵峰，嘴角微微抽动。
他的手放在桌上，手指敲击桌面。
越来越快，越来越重。
最后，拳头砸在桌上。
"啪！"
```

**关键细节**：
- 嘴角抽动
- 手指敲桌面（节奏递进）
- 拳头砸桌
- 声音描写

#### 紧张的表现

```
苏晴坐在沙发上，手指不停地搓着衣角。
她看了一眼钟，又看了一眼门。
来回看了三次。
她的呼吸有些急促，喉结动了动。
```

**关键细节**：
- 搓衣角
- 频频看时间/门
- 动作重复
- 呼吸变化

#### 悲伤的表现

```
顾沉站在窗前，看着远方。
他的手放在窗台上，手指无意识地抓着窗台。
指甲在窗台上划出痕迹。
他没有说话，肩膀微微下沉。
```

**关键细节**：
- 无意识的抓握动作
- 划痕细节
- 肩膀下沉

#### 冷漠的表现

```
赵天坐在主位上，手里拿着茶杯。
他轻轻吹了吹茶水，抿了一口。
然后放下茶杯，发出一声轻响。
他看着前方，眼神没有焦点。
```

**关键细节**：
- 缓慢的动作
- 精致的行为
- 眼神无焦点

### 环境细节库

#### 孤独氛围

```
房间里很安静。
墙上的时钟发出滴答滴答的声音。
窗外，雨淅淅沥沥地下着。
苏晴坐在床边，看着自己的影子。
```

#### 压抑氛围

```
会议室里很闷，空调坏了很久了。
每个人都低着头，不敢说话。
只有赵天的声音在房间里回荡。
窗帘拉得很严，一点阳光都透不进来。
```

#### 温暖氛围

```
阳光透过窗户洒进房间。
空气中弥漫着咖啡的香气。
苏晴坐在沙发上，抱着猫。
猫在她怀里打呼噜。
```

## 平静的狠劲

### 核心原则

**狠在平静中**

要求：
- 不要用激烈的词语
- 不要用夸张的动作
- 用平静的语气和行为
- 通过细节展示狠劲

### 表现技巧

#### 技巧1：对比法

**错误示例**：
```
林风愤怒地说："我要杀了你！"
他狠狠地握紧拳头，眼神凶狠。
```

**正确示例**：
```
赵峰跪在地上，瑟瑟发抖。
林风站在他对面，手里拿着茶杯。
他喝了一口茶，放下。
"赵峰，你不该惹我。"
```

#### 技巧2：微动作

**示例**：
```
林风走到赵峰面前。
他没有看赵峰，只是伸出手。
手指在赵峰的脖子上划过。
"死吧。"
```

#### 技巧3：极简台词

**示例**：
```
"滚。"
顾沉没有抬头，声音很轻。
但没人敢动。
```

#### 技巧4：细节铺垫

**示例**：
```
林风站在尸体旁边，没有表情。
他擦了擦剑上的血，把剑插回鞘中。
"下一个。"
声音很平静，像是在说吃饭一样。
```

### 高级示例

#### 示例1：无声的威胁

```
林风走进房间。
房间里很暗，只有一盏台灯。
赵天坐在沙发后面，看着林风。
林风没有说话，只是走到沙发前。
他坐下，看着赵天。
两人都没有说话，空气很安静。
一分钟后，林风开口。
"赵天。"
"林风。"
"你杀了我父母。"
"是。"
"该轮到你了。"
"你想杀我？"
"不。"
"那你想干什么？"
林风没有回答，只是看着赵天。
他的眼神很平静，像是在看一个死人。
赵天突然感到后背发凉。
```

#### 示例2：平静的杀戮

```
林风站在赵峰面前。
赵峰跪在地上，瑟瑟发抖。
"林风，我错了，我错了..."
林风没有说话。
他拔出剑，剑光如水。
赵峰的说话声停了。
林风把剑插回鞘中。
转身，离开。
从头到尾，没有愤怒，没有激动。
就像在路边踩死一只蚂蚁。
```

#### 示例3：最狠的报复

```
林风站在赵家的废墟上。
他看着这一切，没有表情。
身后，张伟走过来。
"林风，赵家完了。"
"嗯。"
"你开心吗？"
林风没有回答。
他转身，离开。
走到一半，他停下了。
"张伟。"
"在。"
"把赵家的人，都放了。"
"什么？"
"让他们活着。"
"为什么？"
林风回头看了一眼废墟。
"让他们活着，比杀了他们更狠。"
"让他们看着赵家的一切，都没了。"
"让他们每天，都活在绝望里。"
```

## 写作质量检查清单

### 自检清单

- [ ] **模板词检查**：是否有"充满了""坚定""不甘"等模板词？
- [ ] **Show, Don't Tell**：是否用动作/场景代替了解释性叙述？
- [ ] **对话口语化**：对话是否自然、有停顿、有语气词？
- [ ] **细节表现**：情绪是否用具体细节表现？
- [ ] **平静的狠劲**：狠劲是否藏在平静中？

### 常见错误速查

| 错误类型 | 错误示例 | 正确示例 |
|----------|----------|----------|
| 模板化 | 眼中充满了愤怒 | 瞳孔收缩，盯着对方 |
| 直接叙述 | 他很愤怒 | 他一脚踹翻椅子 |
| 对话生硬 | "好的。" | "嗯。" |
| 情绪直述 | 他悲伤地说 | 他的声音发颤 |
| 夸张狠劲 | 愤怒地大吼 | 平静地说"滚" |

## 总结

高级写作技巧的五大核心：

1. **避免模板词** - 用具体动作、场景、对话代替抽象词汇
2. **Show, Don't Tell** - 用动作展示情绪，用场景营造氛围
3. **口语化对话** - 停顿、省略、断句、语气词、口头禅
4. **细节表现情绪** - 动作细节、环境细节、细节组合
5. **平静的狠劲** - 狠在平静中，用微小动作、对比、细节铺垫

**记住**：高级写作，不是用华丽的词藻，而是用真实的细节。
