# 记忆系统初始化检查清单

> 功能1（项目初始化）步骤8的完整执行清单。
> 确保不再遗漏 constraints/ 目录。

## 5 个子目录（缺一不可）

```
[书名]/记忆系统/
├── bible/          ← story_bible.md, character_arcs.md, plot_timeline.md, chekhovs_gun.md
├── summaries/      ← _template.md
├── phases/         ← _template.md
├── volumes/        ← _template.md
└── constraints/    ← cheat_constraints.md, narrative_threads.md, time_constraints.md
```

## 复制模板清单（共 10 个文件）

### bible（从 memory-system/bible/ → 项目）
- [ ] story_bible.md — 填入书名/题材/世界观简版/字数目标
- [ ] character_arcs.md — 空模板，待功能3填充
- [ ] plot_timeline.md — 空模板，待功能4填充
- [ ] chekhovs_gun.md — 空模板

### 其他模板（从 memory-system/ → 项目）
- [ ] summaries/_template.md
- [ ] phases/_template.md
- [ ] volumes/_template.md

### 强约束系统（从 templates/constraints/ → 项目）
- [ ] cheat_constraints.md — 填入金手指基本信息（名称/类型/限制）
- [ ] narrative_threads.md — 填入已知叙事线（主线/感情线等）
- [ ] time_constraints.md — 填入时间基准

## ⚠️ 常犯错误

- **漏创建 constraints/ 目录**：功能1的模块文件 `modules/05_01_功能模块.md` 步骤8.1只列了4个目录，必须补上第5个
- **漏复制约束模板**：步骤8.2b只复制了 summaries/phases/volumes 模板，必须追加 constraints 下3个模板
- **初始化报告只列4项**：完成报告应列出 bible×4 + constraints×3 共7项
