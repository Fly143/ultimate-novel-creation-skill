# 功能模块 21：批量生成设定

**功能**：批量生成配角人设、支线剧情、反派角色等短文本设定内容。

> ⚠️ **不含批量写章节**。LLM 有输出上限，批量写章节不可行。章节续写请用场景 B（逐章续写 + 记忆系统接力）。

---

## 批量生成配角人设

### 使用方式

```
输入：批量生成10个配角人设
输入：生成5个宗门长老
输入：设计3个反派角色
```

### 执行流程

```
1. 确认生成数量和类型
2. 读取世界设定 + 主角人设（确保配角与世界观一致、颜值不超主角）
3. 一次性生成所有人设（每条约 200-400 字）
4. 保存到 [书名]/设定/配角人设_批量.md
5. 更新记忆系统（如已启用）
```

---

## 批量生成支线剧情

### 使用方式

```
输入：生成5条支线剧情
输入：设计3条感情线
```

### 执行流程

```
1. 确认数量和类型
2. 读取主线大纲，确保支线不与主线冲突
3. 一次性生成所有支线（每条 200-500 字）
4. 保存到 [书名]/设定/支线剧情_批量.md
5. 更新情节大纲中的支线标记
```

---

## 批量生成反派/组织

### 使用方式

```
输入：设计3个反派组织
输入：生成5个敌对势力
```

### 执行流程

```
1. 读取世界观设定，确保组织与力量体系一致
2. 一次性生成（每个 300-500 字，含背景/动机/实力）
3. 保存到 [书名]/设定/反派组织_批量.md
4. 更新 story_bible.md 中的势力格局
```

---

## 批量生成报告

完成批量生成后输出：

```
=== 批量生成完成报告 ===

【生成统计】
- 类型：[人设/支线/反派/...]
- 数量：X 项
- 保存位置：[书名]/设定/xxx_批量.md

【内存占用】
- 设定文件大小：约 X KB
- 记忆系统已同步：是/否
```

---

## 注意事项

- 单次不超过 10 项，避免输出截断
- 仅生成设定/规划类内容，不包含正文章节
- 批量写章节请用场景 B（继续创作）：说「写第 X 章」→ 等完成 → 说「继续」→ 下一章
- 生成后建议人工抽查质量
