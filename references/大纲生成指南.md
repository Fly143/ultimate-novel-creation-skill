# 大纲生成指南

## 大纲的层次结构

### 【一句话概括】（20字以内）
核心要素：主角+冲突+目标+亮点
示例：穿越者重生1990，用先知优势打造商业帝国，打脸仇敌

### 【核心设定】
- **世界观设定**：时代背景/地点/力量体系
- **核心金手指**：系统/重生/特殊能力
- **主角身份**：初始身份和目标
- **核心冲突**：主角面临的最大障碍

### 【人物小传】
- **主角**：姓名、年龄、性格、金手指、目标
- **关键配角**：盟友、对手、CP等
- **反派**：主要反派的目标和手段

### 【主线剧情】
按照时间线或阶段划分
- **开篇（1-10章）**：主角登场，金手指觉醒/发现
- **发展（11-50章）**：小高潮，初显实力
- **高潮（51-100章）**：大事件，打脸反派
- **上升（101-200章）**：新地图，新挑战
- **终极（200+章）**：最终对决，达成目标

### 【爽点分布】
每个阶段的核心爽点
- 开篇：觉醒金手指
- 发展：初露锋芒
- 高潮：打脸反派
- 上升：突破境界
- 终极：登顶巅峰

### 【关键剧情点】
5-8个改变故事走向的重大事件
- 金手指觉醒
- 初遇盟友/CP
- 首次打脸
- 被反派压制
- 反击高潮
- 揭露真相
- 最终决战

## 不同题材的大纲模板

### 现言/总裁文
```
一句话概括：
[主角]穿越/重生/误打误撞，成为[身份]，与[男主]相爱相杀，最终HE

核心设定：
- 地点：现代都市
- 身份：普通人/豪门千金/替嫁新娘等
- 金手指：系统/先知/特殊能力/无金手指
- 核心冲突：身份差距/家族反对/前任阻碍

主线剧情：
开篇（1-20章）：女主穿越/重生，初次相遇
发展（21-60章）：感情升温，误会产生
高潮（61-100章）：误会解开，家族反对
上升（101-200章）：共同面对危机，感情加深
终极（200+章）：终成眷属，甜蜜结局

爽点分布：
- 豪门生活的反差
- 打脸前任和绿茶
- 男主的宠溺和保护
- 女主的逆袭和成长
```

### 古言/穿越文
```
一句话概括：
[主角]穿越到[朝代/地点]，成为[身份]，与[男主]相爱相杀，最终HE

核心设定：
- 地点：古代宫廷/江湖/朝堂
- 身份：公主/千金/平民/宫女等
- 金手指：现代知识/系统/特殊能力
- 核心冲突：身份差距/宫廷斗争/家族利益

主线剧情：
开篇（1-20章）：穿越，适应环境，初遇男主
发展（21-60章）：逐步融入，感情萌芽
高潮（61-100章）：危机爆发，情感升温
上升（101-200章）：宫廷斗争，共同面对
终极（200+章）：终成眷属，执掌朝堂/江湖

爽点分布：
- 现代知识在古代的碾压
- 宫廷斗争的逆袭
- 男主的深情守护
- 女主的聪明才智
```

### 玄幻/修仙文
```
一句话概括：
[主角]穿越/重生到[世界]，获得[金手指]，从[境界]修炼到[境界]，最终成神

核心设定：
- 世界：玄幻世界/仙侠世界
- 境界：炼气/筑基/金丹/元婴等
- 金手指：系统/特殊体质/神器
- 核心冲突：弱肉强食，资源争夺

主线剧情：
开篇（1-50章）：穿越/重生，觉醒金手指，初步修炼
发展（51-150章）：初显实力，结识盟友，打脸对手
高潮（151-300章）：突破境界，大事件，击杀强敌
上升（301-500章）：新地图，新挑战，更强敌人
终极（500+章）：最终境界，成神/成帝

爽点分布：
- 境界突破的爽感
- 打脸反派的畅快
- 获得宝物的惊喜
- 炼丹炼器的成就感
```

### 科幻/系统文
```
一句话概括：
[主角]绑定[系统]，从[身份]成长为[身份]，解决[危机]，最终[结局]

核心设定：
- 世界：未来世界/末世/星际
- 系统：系统类型和功能
- 核心冲突：生存危机/资源短缺/敌人威胁

主线剧情：
开篇（1-50章）：绑定系统，了解功能，初试身手
发展（51-150章）：积累资源，建立势力，应对危机
高潮（151-300章）：大危机，系统升级，力挽狂澜
上升（301-500章）：新危机，新挑战，更强敌人
终极（500+章）：终极危机，拯救世界

爽点分布：
- 系统功能的强大
- 快速成长的爽感
- 建立势力的成就感
- 拯救世界的荣耀
```

## 大纲优化建议

### 检查清单
- [ ] 一句话概括是否精准（20字以内）
- [ ] 核心设定是否清晰
- [ ] 人物小传是否完整
- [ ] 主线剧情是否连贯
- [ ] 爽点分布是否均匀
- [ ] 关键剧情点是否突出

### 优化技巧
1. **核心冲突前置**：开篇就要展示核心冲突
2. **爽点节奏**：每3-5章一个小高潮
3. **伏笔埋设**：关键情节要有伏笔
4. **人物成长**：每个阶段都要有成长
5. **主题升华**：从个人成长到价值追求

### 常见问题
1. **主线不清晰**：多个故事线混杂
2. **爽点不足**：缺乏高潮和亮点
3. **人物弱化**：配角盖过主角
4. **逻辑矛盾**：前后设定不一致
5. **节奏拖沓**：铺垫过长，高潮乏力

## 大纲生成步骤

### 第一步：确定核心要素
- 题材类型
- 主角身份
- 核心金手指
- 核心冲突
- 目标和结局

### 第二步：构建世界观
- 时代背景
- 力量体系
- 社会结构
- 特殊规则

### 第三步：设计人物
- 主角性格和成长
- 配角和CP
- 反派设定
- 人物关系网

### 第四步：规划主线
- 划分阶段
- 标注爽点
- 设置高潮
- 安排转折

### 第五步：填充细节
- 关键剧情点
- 重要配角
- 伏笔和反转
- 主题升华
