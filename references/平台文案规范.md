# 平台文案规范

## 番茄小说平台

### 用户画像
- 18-35岁为主
- 快节奏阅读习惯
- 偏好强冲突、强爽感
- 对甜宠、打脸、逆袭类题材接受度高

### 文案结构
```
【黄金开头】（3秒抓住读者）
- 核心冲突前置
- 强烈的反差或悬念
- 主角的困境或优势

【核心看点】（为什么读这本）
- 爽点类型：打脸/逆袭/甜宠/复仇
- 金手指：系统/重生/特殊能力
- 情感线：甜宠/虐恋/破镜重圆

【期待钩子】（为什么要继续读）
- 悬念设置
- 伏笔暗示
- 下一个爽点预告

【关键词】（平台偏好）
- 宠妻、打脸、逆袭、系统、重生、甜宠
```

### 示例
```
《重生1990：从摆摊到首富》

【简介】
被渣男和小三害死，我重生回到了1990年！
这一次，我要夺回属于我的一切！
从小小摆摊摊主开始，建立商业帝国，让那些欺负我的人跪地求饶！
没想到，那个在商场上叱咤风云的冰山总裁，竟然是我的青梅竹马？
他红着眼把我堵在墙角："跟我回家，我的家产都是你的。"

【标签】
重生 | 逆袭 | 打脸 | 甜宠 | 豪门
```

## 晋江文学城

### 用户画像
- 女性读者为主
- 注重情感和人物塑造
- 偏好细腻的情感描写
- 对剧情逻辑和人物成长有要求

### 文案结构
```
【故事核心】（简要介绍）
- 人物关系
- 核心冲突
- 故事主题

【情感基调】（情感氛围）
- 甜度/虐度
- 情感走向
- 人物成长

【看点亮点】（为什么值得读）
- 独特设定
- 精彩情节
- 人物魅力

【风格说明】
- 文风特点
- 节奏把控
- 结局类型
```

### 示例
```
《高冷校草的软萌学霸》

【文案】
林小雨，A大数学系的学霸，外表软萌，内心坚韧。
顾言，A大计算机系的系草，高冷孤傲，不近女色。

一次意外的图书馆相遇，让两个本不相交的人产生了交集。
"同学，这本书是我的。"
"哦，抱歉。"

"等等，你在看什么？"
"《数论基础》，有问题吗？"
"你...看得懂？"

从此，高冷校草的日常就是：
"小雨，今天一起吃饭吗？"
"小雨，这道题怎么做？"
"小雨，做我女朋友吧。"

【标签】
校园 | 甜宠 | 学霸 | 欢喜冤家 | 1V1 | HE
```

## 起点中文网

### 用户画像
- 男性读者为主
- 注重世界观和设定
- 偏好硬核内容和升级流
- 对逻辑和战力体系要求高

### 文案结构
```
【世界观简介】（设定背景）
- 时代/地点
- 力量体系
- 核心规则

【主角设定】（为什么能成神）
- 身份背景
- 金手指/特殊能力
- 成长路径

【核心看点】（爽点来源）
- 逆袭打脸
- 升级突破
- 探索冒险

【预期篇幅】（给读者预期）
- 字数范围
- 更新频率
- 完结保证
```

### 示例
```
《我在末日种田》

【简介】
末日第三年，别人在抢罐头，我在种番茄。
不是我不想战斗——是我的系统太奇葩了。
别人觉醒的是火焰异能、雷电异能。
我觉醒的是"农业精通"。

丧尸来了怎么办？
——先施个肥吧。

这是一个末日种田的故事。
不太热血，不太正经。
但番茄是真的甜。

【数据】
总字数：180万+
更新：每日2更，每更2500字
状态：连载中

【标签】
末世 | 种田 | 系统 | 轻松 | 搞笑
```

## POPO小说

### 用户画像
- 25-40岁女性为主
- 追求情感共鸣和代入感
- 偏好甜宠、虐恋、悬疑
- 对情感深度和人物心理有要求

### 文案结构
```
【情感钩子】（第一眼吸引）
- 强烈的情感冲突
- 悬念设置
- 反差萌

【人物设定】（让人记住的角色）
- 男主：性格、身份、魅力点
- 女主：性格、身份、成长
- CP关系：为什么搭

【情感线】（核心看点）
- 情感走向
- 矛盾冲突
- 甜虐比例

【标签】（精准定位）
- 甜宠/虐恋/破镜重圆
- 1V1/多CP/无CP
- HE/BE/开放式
```

### 示例
```
《霸道总裁的替嫁新娘》

【简介】
"嫁给他，你妹妹的医药费我全包了。"
一场交易，她替妹出嫁，成了他的名义妻子。

"记住，不要爱上我。"他冷冷警告。
"放心，我只认钱。"她云淡风轻。

但当家族危机来临，他把她护在身后时，她才意识到——
这个只谈利益的男人，早就动了心。

【标签】
总裁 | 替嫁 | 甜宠 | 破镜重圆 | 1V1 | HE
```

## 文案优化建议

### 避免踩坑
1. 不要脱离平台调性
2. 不要过于冗长
3. 不要忘记核心看点
4. 不要忽视情感钩子
5. 不要忽略标签设置

### 优化技巧
1. **黄金3秒**：开头就要抓住读者
2. **冲突前置**：把最激烈的冲突放在前面
3. **悬念设置**：让读者想继续读下去
4. **情感共鸣**：让读者有代入感
5. **标签精准**：符合平台推荐机制

### A/B测试
如果不确定，可以准备2-3个文案版本：
- 测试不同开头方式
- 测试不同悬念设置
- 测试不同情感钩子
- 根据数据反馈选择最佳方案
