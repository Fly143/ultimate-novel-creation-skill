# 高级写作技巧

## 模板词：你需要杀掉的 15 个词

以下词一旦出现，删掉重写。不是「改一个词」，是整句重写。

| 删掉 | 换成 |
|------|------|
| 眼中充满了愤怒 | 瞳孔收缩、盯着对方不动、手搭上剑柄 |
| 冷冷地说 | 删掉「冷冷地」，让对话本身冷。或：头也没抬，声音从牙缝出来 |
| 紧紧地握住 | 指节发白、青筋暴起、指甲嵌进掌心 |
| 愤怒地 / 悲伤地 / 激动地说 | 全删。用动作替代。或者什么都不加——让对话本身撑住 |
| 难以置信 | 删掉。展示「难以置信」的行为：后退半步、摇头、说不出话 |
| 狠狠地 / 猛地 / 默默地 | 删掉副词。动词本身够硬就不需要副词。打就是打，不需要「狠狠地打」 |
| 充满了 | 删掉。不说「眼中充满了悲伤」，说「眼眶红了，眼角有水痕」 |

## Show, Don't Tell：三种场景速查

**情绪 → 动作**：不说「他很愤怒」，说「他一脚踹翻椅子，喘着粗气」

**氛围 → 环境**：不说「这是个悲伤的夜晚」，说「窗外下雨，灯关着，只有手机屏幕的微光和她轻微的抽泣声」

**关系 → 互动**：不说「他们感情深厚」，说「两人对视一秒，都笑了。他伸出手，她握住」

## 对话：像活人说话

- 用停顿：「我不会……放过你的。」
- 用省略：「我真的很喜欢你……」后半句说不出口比说出来有力
- 用短句：「嗯。」比「好的，我明白了」真实十倍
- 口头禅藏在性格里——霸道的人只说「嗯」「好」「滚」，傲娇的人死不说「喜欢」

---

## 平静的狠劲

最高级的狠不在嘶吼里，在平静里。

不要在复仇时写「我要杀了你！」，让他喝一口茶，放下杯子，说「赵峰，你不该惹我」。

不要在决胜时写「去死吧！」，让他擦剑上的血，插回鞘中，说「下一个」。声音平静得像在说吃饭。

平静的暴力比嘶吼的暴力可怕一百倍。因为读者知道——这个人不是情绪失控，他是真的、冷静地、要你的命。

## 自检

写完每章后扫一眼：
- [ ] 有没有「充满了」「坚定」「不甘」？
- [ ] 有没有「XX地说」？
- [ ] 情绪是用动作/场景表现的，还是直接说的？
- [ ] 对话像活人说的吗？


## 反 AI 铁律

以下规则在每章正文中强制执行，用于去除「AI 味」：

**铁律一**：叙述者不替读者下结论。读者能从动作推断的，不要写出来。
- ✗ 「他意识到这是人生最重要的一战」
- ✓ 直接写那场战斗——让赌注自己说话

**铁律二**：正文中禁用分析/报告语言。以下词永远不出现：
核心动机、信息不对称、战略优势、最优解、值得注意、关键要点

**铁律三**：AI 高频词限流（每 3000 字最多 1 个）：
画卷、见证、错综复杂、至关重要、充满活力、踏上、全面、细致入微、景观（比喻义）、领域（比喻义）、促进、强调

**铁律四**：禁止比喻重复。同一个意象出现两次后，第三次必须换。

**铁律五**：禁止「不是 X，而是 Y」句式。每章最多 1 次。
- ✗ 「那不是恐惧。那是更深层的东西。」
- ✓ 直接写那个东西

**铁律六**：禁止描写中排比三个形容词。每 2000 字最多 1 次。
- ✗ 「古老、可怕而浩瀚」
- ✓ 用一对或单个精准的词

**铁律七**：情绪不贴标签，用动作外化。
- ✗ 「他感到一阵愤怒」
- ✓ 「他一掌拍在桌上。水杯倒了。」
