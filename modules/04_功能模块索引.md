## 功能分类速查

### 立项类（功能1）
- **功能1**：项目初始化 → 生成书名、创建立项定位单、推荐创作模式、可选初始化记忆系统

### 设定类（功能2-3）
- **功能2**：世界观构建 → 设定世界规则、力量体系、社会结构
- **功能3**：人物设计 → 创建人物卡片、人物关系图谱

### 大纲类（功能4）
- **功能4**：情节大纲 → 生成故事主线、章节规划表

### 创作类（功能5-11）
- **功能5**：黄金开篇创作 → 生成前三章黄金开篇
- **功能6**：单章写作 → 生成单章内容（核心写作功能，集成记忆系统）
- **功能7**：爽点设计 → 设计爽点桥段
- **功能8**：冲突与反转设计 → 设计冲突和反转情节
- **功能9**：对话场景创作 → 生成对话为主的场景
- **功能10**：打斗场景创作 → 生成打斗场景
- **功能11**：AI/系统流创作 → 生成AI/系统流特殊场景

### 诊断类（功能12-17）
- **功能12**：小说节奏诊断与优化 → 分析节奏问题并优化
- **功能13**：章节质量检查 → 检查章节质量（自动触发）
- **功能14**：创新深度检查 → 检查创新深度（自动触发）
- **功能15**：商业可行性评估 → 评估商业潜力（完结触发）
- **功能16**：修订优化 → 修订和优化已有内容

## 使用指南

**当用户选择某个功能时**：
1. 根据功能编号，在上表中找到对应的模块文件路径
2. 使用 `read_file` 工具读取对应的模块文件
3. 按模块文件中的流程执行功能

**示例**：
- 用户选择"单章写作" → 读取 `modules/05_06_功能模块.md`
- 用户选择"人物设计" → 读取 `modules/05_03_功能模块.md`
- 用户选择"记忆系统管理" → 读取 `modules/05_18_功能模块.md`
- 用户说"看记忆" → 读取 `modules/05_18_功能模块.md`，定位到工作流B
- 用户说"人物跑偏了" → 读取 `modules/05_18_功能模块.md`，定位到工作流F
- 用户选择"强约束系统" → 读取 `modules/05_22_功能模块.md`
- 用户说"金手指"/"时间约束"/"叙事线" → 读取 `modules/05_22_功能模块.md`，定位到对应子系统
- 用户说"分析《XXX》的YYY" → 读取 `modules/05_23_功能模块.md`，子功能A
- 用户说"分析《XXX》的风格" → 读取 `modules/05_23_功能模块.md`，子功能B
- 贴原文+用户说"分析这个人物" → 读取 `modules/05_23_功能模块.md`，子功能A（原文模式）
- 用户说"读者审稿"/"审稿"/"审核剧情单元" → 读取 `modules/05_25_功能模块.md`
- 用户说"版本控制"/"回滚"/"查看历史" → 读取 `modules/05_21_功能模块.md`

## 重要说明

### 记忆系统使用方式（v5.0 自然语言记忆）

**初始化时机：**
- 在功能1（项目初始化）中，可选择是否初始化记忆系统
- 强烈推荐长篇连载（>20万字）使用

**自动维护：**
- 在功能6（单章写作）中，记忆系统作为内置流程自动维护
- 每章写完后自动更新 5 个圣经文档 + 单章摘要
- 每 10 章自动执行阶段压缩

**手动管理：**
- 在功能18（记忆系统管理）中，可进行以下操作：
  - 工作流A：项目初始化（创建 [书名]/记忆系统/ 目录及圣经模板）
  - 工作流B：查看记忆（显示当前创作状态/人物/剧情/伏笔）
  - 工作流C：手动编辑圣经（更新人设/剧情/伏笔）
  - 工作流D：阶段压缩（手动触发 10 章压缩）
  - 工作流E：跨会话恢复（新会话续写时调用）
  - 工作流F：紧急修复跑偏（检查 OOC/剧情矛盾）

**推荐使用策略：**
- **日常写作**：使用功能6（单章写作），记忆系统自动维护
- **首次初始化**：使用功能1（项目初始化）或功能18工作流A
- **查看状态**：使用功能18工作流B
- **紧急修复**：使用功能18工作流F
- **跨会话续写**：系统自动调用工作流E
