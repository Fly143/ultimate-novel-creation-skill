# CLI 调用指南

> 本文件说明如何在 Hermes 中正确加载和使用中文名技能。

---

## 问题背景

Hermes v0.14.0 中，中文名技能（如"全能小说作家"）不会自动注册斜杠命令（如 `/全能小说作家`）。这是因为中文字符在命令解析中可能被截断或误识别。

---

## 正确的加载方式

### 方式1：对话触发（推荐）

在对话中直接说技能的触发词，Hermes 会自动识别并加载：

| 触发词 | 效果 |
|--------|------|
| 写小说 / 新开一本 | 加载技能，进入新开小说流程 |
| 继续 / 下一章 / 接着写 | 加载技能，继续创作 |
| 书名 / 人设 / 大纲 | 加载技能，使用对应功能 |
| 菜单 / 帮助 | 加载技能，显示功能列表 |

### 方式2：技能名称触发

直接说技能名称"全能小说作家"或"小说创作"，Hermes 会识别并加载。

### 方式3：skill_view 命令

如果需要查看技能内容，可以使用：

```
skill_view(name="全能小说作家")
```

---

## 注意事项

1. **不需要斜杠命令**：中文名技能不依赖斜杠命令，通过对话触发即可
2. **触发词优先**：技能会监听特定的触发词，说这些词就能激活
3. **首次交互**：首次加载技能时，会执行智能识别流程（读取 00_强制执行协议.md）
4. **跨会话**：技能在新会话中需要重新触发（说触发词）

---

## 常见问题

**Q：为什么我说"全能小说作家"没有反应？**
A：确保 Hermes 已加载该技能。如果没有，尝试说"加载全能小说作家"或直接说触发词（如"写小说"）。

**Q：斜杠命令能用吗？**
A：中文名技能不注册斜杠命令。请使用对话触发方式。

**Q：怎么知道技能已加载？**
A：加载后，Hermes 会执行首次交互流程（意图识别），并根据你的输入进入对应场景。
